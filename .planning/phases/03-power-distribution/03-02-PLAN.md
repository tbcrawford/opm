<task type="auto">
  <name>Task 1: Version injection in main.go and cmd/root.go</name>
  <files>main.go, cmd/root.go</files>
  <action>
    Update `main.go` to add version vars and pass them to the cmd package:

    ```go
    package main

    import "github.com/tbcrawford/opm/cmd"

    // version and commit are injected at build time by GoReleaser via ldflags:
    //   -X main.version={{.Version}} -X main.commit={{.Commit}}
    var (
        version = "dev"
        commit  = "none"
    )

    func main() {
        cmd.SetVersionInfo(version, commit)
        cmd.Execute()
    }
    ```

    Add `SetVersionInfo` to `cmd/root.go`:
    ```go
    // SetVersionInfo sets the version string displayed by `opm --version`.
    // Called from main() with values injected by GoReleaser ldflags.
    func SetVersionInfo(version, commit string) {
        rootCmd.Version = version + " (" + commit + ")"
    }
    ```

    This gives: `opm version v0.1.0 (abc1234)` when built by GoReleaser.
    When built locally: `opm version dev (none)`.

    Run `go build ./... && ./opm --version` to verify output.
  </action>
  <verify>
    <automated>go build -o opm . && ./opm --version</automated>
  </verify>
  <done>`opm --version` prints version string; SetVersionInfo wired in main.go and cmd/root.go.</done>

<task type="auto">
  <name>Task 2: .goreleaser.yaml</name>
  <files>.goreleaser.yaml</files>
  <action>
    Create `.goreleaser.yaml`:

    ```yaml
    # GoReleaser v2 configuration for opm
    # https://goreleaser.com
    version: 2

    project_name: opm

    before:
      hooks:
        - go mod tidy
        - go test ./...

    builds:
      - id: opm
        main: .
        binary: opm
        env:
          - CGO_ENABLED=0
        goos:
          - darwin
          - linux
        goarch:
          - amd64
          - arm64
        ldflags:
          - -s -w
          - -X main.version={{.Version}}
          - -X main.commit={{.Commit}}
        mod_timestamp: "{{ .CommitTimestamp }}"

    # Merge darwin amd64 + arm64 into a single universal binary
    universal_binaries:
      - id: opm
        replace: true

    archives:
      - id: opm
        formats:
          - tar.gz
        format_overrides:
          - goos: windows
            format: zip
        name_template: "{{ .ProjectName }}_{{ .Version }}_{{ .Os }}_{{ .Arch }}"
        files:
          - LICENSE
          - README.md

    checksum:
      name_template: "checksums.txt"
      algorithm: sha256

    changelog:
      sort: asc
      filters:
        exclude:
          - "^docs:"
          - "^test:"
          - "^chore:"
          - Merge pull request
          - Merge branch

    # Homebrew tap — requires GH_PAT secret with repo scope on tbcrawford/homebrew-tap
    # Prerequisites: create the tbcrawford/homebrew-tap repo on GitHub before first release.
    brews:
      - name: opm
        repository:
          owner: tbcrawford
          name: homebrew-tap
          token: "{{ .Env.GH_PAT }}"
        directory: Formula
        homepage: "https://github.com/tbcrawford/opm"
        description: "OpenCode profile manager — switch configs like docker context"
        license: MIT
        test: |
          system "#{bin}/opm", "--version"
        install: |
          bin.install "opm"
    ```
  </action>

  </verify>
  <done>.goreleaser.yaml written with correct v2 syntax; builds, universal_binaries, archives, checksum, brews sections all present.</done>

<task type="auto">
  <name>Task 3: GitHub Actions workflows and .gitignore</name>
  <files>.github/workflows/release.yml, .github/workflows/ci.yml, .gitignore</files>
  <action>
    Create `.github/workflows/release.yml`:

    ```yaml
    name: Release

    on:
      push:
        tags:
          - "v*"

    permissions:
      contents: write

    jobs:
      release:
        name: Release
        runs-on: ubuntu-latest
        steps:
          - name: Checkout
            uses: actions/checkout@v4
            with:
              fetch-depth: 0

          - name: Set up Go
            uses: actions/setup-go@v5
            with:
              go-version: stable

          - name: Run GoReleaser
            uses: goreleaser/goreleaser-action@v6
            with:
              distribution: goreleaser
              version: "~> v2"
              args: release --clean
            env:
              GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
              # Required to push the Homebrew formula to tbcrawford/homebrew-tap.
              # Create a PAT with 'repo' scope and add it as GH_PAT in repo secrets.
              GH_PAT: ${{ secrets.GH_PAT }}
    ```

    Create `.github/workflows/ci.yml`:

    ```yaml
    name: CI

    on:
      push:
        branches: [main]
      pull_request:
        branches: [main]

    jobs:
      test:
        name: Test
        runs-on: ${{ matrix.os }}
        strategy:
          matrix:
            os: [ubuntu-latest, macos-latest]
            go: [stable]
        steps:
          - name: Checkout
            uses: actions/checkout@v4

          - name: Set up Go
            uses: actions/setup-go@v5
            with:
              go-version: ${{ matrix.go }}
              cache: true

          - name: Test
            run: go test ./... -v -count=1

          - name: Vet
            run: go vet ./...

          - name: Build
            run: go build ./...
    ```

    Create `.gitignore`:
    ```
    # GoReleaser build output
    dist/

    # Go build artifacts
    opm
    *.test

    # Editor
    .DS_Store
    ```
  </action>
  <verify>
    <automated>go build ./... && echo "build ok"</automated>
  </verify>
  <done>All three files created; workflows use goreleaser-action@v6 and actions/setup-go@v5; .gitignore excludes dist/.</done>
