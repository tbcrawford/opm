OpenCode Profile Manager

Switch contexts.
Instantly.

Manage multiple isolated OpenCode environments — MCPs, agents, models, plugins — and switch between them with a single command.

Get Started View on GitHub
opm — zsh
# migrate your existing config
$ opm init
✓ Migrated ~/.config/opencode → profile "default"
# create a new isolated profile
$ opm create work
✓ Created profile "work"
# switch — takes effect immediately
$ opm use work
✓ Switched to context "work"
Quick Start

Up and running
in minutes.

Install
brew install tbcrawford/tap/opm
Then three commands and you're set
01
Initialize
Migrate your existing OpenCode config non-destructively into a named profile.
opm init
02
Create a profile
Create a new isolated environment with its own MCPs, agents, and model config.
opm create work
03
Switch
Swap to your new profile instantly. No restarts required.
opm use work
Why opm

Fully isolated.
Zero configuration.

⚡
Instant switching
One command swaps the active profile via directory-level symlink. No restarts. Takes effect immediately.
🔒
Complete isolation
Each profile is an independent directory — separate MCPs, agents, plugins, AGENTS.md, and model configs.
🔀
Non-destructive init
opm init migrates your current config safely. Your existing setup becomes the "default" profile, untouched.
🐳
docker context UX
Familiar command surface if you've used docker context. Create, use, ls, inspect, rm — it all maps naturally.
Reference

Commands

Command
Description
opm init
Migrate existing config into opm. Safe to run on an existing setup.
opm use <name>
Switch to a profile.
opm create <name>
Create a new empty profile.
opm list
List all profiles, showing which is currently active.
opm show
Print the name of the currently active profile.
opm inspect <name>
Show detailed information about a profile.
opm rename <old> <new>
Rename an existing profile.
opm copy <src> <dst>
Copy a profile to a new name.
opm remove <name> [name...]
Remove one or more profiles.
opm path <name>
Print the filesystem path to a profile directory.
opm reset
Restore ~/.config/opencode to a plain directory.
opm doctor
Check opm installation health.
